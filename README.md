# 🪨 Stone Paper Scissors Game (Python)

A simple command-line **Stone, Paper, Scissors** game built using Python. Play against the computer and test your luck!

---

## 🎮 Features

* Menu-driven game
* User vs Computer gameplay
* Random computer choices using `random` module
* Input validation
* Infinite loop until user exits

---

## 🧠 Concepts Used

* Python loops (`while`)
* Conditional statements (`if-elif-else`)
* Lists
* User input handling
* Random module

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/stone-paper-scissors.git
```

2. Navigate to the folder:

```bash
cd stone-paper-scissors
```

3. Run the program:

```bash
python game.py
```

---

## 🕹️ How to Play

* Choose an option:

  * `1` → Stone
  * `2` → Paper
  * `3` → Scissors
  * `4` → Exit

* The computer will randomly select its choice.

* The winner is decided based on standard game rules:

  * Stone beats Scissors
  * Scissors beats Paper
  * Paper beats Stone
